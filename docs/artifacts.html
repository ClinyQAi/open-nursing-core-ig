These define constraints on FHIR resources for systems conforming to this implementation guide.

Assessment of the patient's housing status.

Assessment of fall risk using the Morse Fall Scale.

ONC NHS Patient

An NHS-specific patient profile that mandates the inclusion of ethnicity data for health equity analysis.

Observation for Fitzpatrick Skin Type to support equity in assessment interpretation.

Base profile for nursing assessments.
